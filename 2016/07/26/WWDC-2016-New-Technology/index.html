WWDC 2016 有哪些新技术

发表于 2016-07-26 |

New API

语音识别

@see SFSpeechRecognizer

新的 SFSpeechRecognizer 框架，可以分析语音内容，支持语音转文字。

SirKit

@see Programming Guide
@see Intents.framework
@see IntentsUI.framework
@see 使用 Swift 在 iOS 10 中集成 Siri —— SiriKit 教程（Part 1）
@see 使用 Swift 在 iOS 10 中集成 Siri —— SiriKit 教程（Part 2）

借助SiriKit，应用能够识别具体的静态或者动态的词汇。借助 interpreted meta-data（Intent）Siri 能够调用你的应用，使用一些功能。

New通知框架

@see UserNotifications和 UserNotificationsUI
@see WWDC2016-session707 Introduction to Notifications(简书Neal)
@see wwdc2016-session707 Notifications(淅沥枫)
@see WWDC2016 Session笔记 - iOS 10 推送Notification新特性(一缕殇流化隐半边冰霜)

  • 统一本地和远程通知
  • 在通知里可以加入多媒体内容，比如图片和 GIF 图片
  • 当用户禁止通知时，你会收到告知
  • 能够获得你应用里的通知清单，能够控制触发时间，能更新这些通知，甚至具备删除旧通知的能力
  • 通过代码可以获取用户的通知设置情况

CloudKit

@see CloudKit
支持 CloudKit 的分享操作，会显示一个界面让用户浏览和修改参与者、开始分享和结束分享。
@see UICloudSharingController 和 UICloudSharingControllerDelegate
还可以在 iOS 10 的 Notes.app 看到

Foundation

@see Foundation Release Notes for OS X v10.12 and iOS 10
这个版本带来了许多持续API改进基础上,利用新的API特性在objective - c中,使API在Swift更自然。

String 类型的枚举

@see Using Swift with Cocoa and Objective-C (Swift 3)

  • NS_STRING_ENUM

  • NS_EXTENSIBLE_STRING_ENUM

Objective-C Class Properties


单位转换

新的 API，能够转换各种不同的测量单位。

  • NSUnit - The abstract superclass of units, such as miles, km/h, degrees Fahrenheit.
  • NSDimension - Subclass of NSUnit representing unit families, such as length, speed, temperature. This class has many subclasses such as NSUnitLength, representing the different unit families.
  • NSUnitConverter - Helper class to make it easy to convert between units in a unit value.
  • NSMeasurement - A unit and a value, representing a measured quantity. For instance 10 miles, 42 km/h, 72 degrees Fahrenheit.
  • NSMeasurementFormatter - A new formatter to display measurements in localized fashion.

NSURL

NSURL对象是不可变的,必须有一个URL字符串。所以[NSURL init]没有意义,因为没有提供URL字符串。之前的版本中,[NSURL init]会released self ,返nil。

新版本中[NSURL init],回返回一个空字符串。[NSURL new]也是一样的。

运行多个iOS模拟器可以导致NSUserDefaults不起作用

模拟器运行iOS 8或9其次是iOS 10模拟器将导致模拟器NSUserDefaults停止工作。需要重启Mac来解决。

日期

  • NSISO8601DateFormatter 出现在 Foundation 里了
  • 有了新的 NSDateInterval 类型

对 API 做去字符串化（De-stringification）

  • NSCalendarIdentifier 现在已经是一个专门的类型了（代替了 NSString）

  • NSError.domain 现在是一个 NSErrorDomain，代替了 NSString

  • 通知里使用 NSNotificationName 代替 NSString

  • stringByApplyingTransform:reverse: 使用 NSStringTransform 类型代替 NSString

  • NSRunLoop.mode 现在是一个 NSRunLoopMode

  • NSHTTPCookie 属性现在是 [NSHTTPCookiePropertyKey: AnyObject]，代替了 [NSString: AnyObject]

  • 等等

新的日志 API

NSLog is now just a shim to os_log in most circumstances.

New timer API


现在可以传入block（Xcode 8 的 Swift 还不支持这个功能，会在 Swift 3 加入。）
这个需求在开源社区中早已广为流传，这次苹果将其吸收。

NSURLSession

新的委托方法允许你获取一些网络相关的数据。提供了很多元数据，比如不同场景的开始和结束时间（DNS 查询、连接……）
@see NSURLSessionTaskMetrics 和 NSURLSessionTaskTransactionMetrics
@see WWDC 2016 session 711 NSURLSession new features and best practices

UITableView 和 UICollectionView 预加载

@see WWDC 2016 session 219 What’s New in UICollectionView in iOS 10
@see WWDC16笔记：What’s new in UICollectionView(ynboh)
@see WWDC2016 Session笔记 - iOS 10 UICollectionView新特性

  • 预加载：在到达滑动区域的边缘之前预先下载数据，更轻松地实现多页内容。
  • UIRefreshControl 目前已经可以在 UIScrollView 上使用了（之前只能在 UITableViewController 上使用）。
  • 新的 UICollectionViewFlowLayoutAutomaticSize CollectionView 约束条件。

GCD

@see WWDC 2016 session 720 Concurrent Programming With GCD in Swift 3

新的 Dispatch API 出现在 Swift 3 里，用于 GCD（Grand Central Dispatch）。在 Swift 3 里操作 GCD 更容易也更美观

UUID 值类型

@see NSUUID
在 Swift 3 里出现了一个新的类型 UUID，是一个值类型（估计和 Objective-C 里的 NSUUID 类似）。

UIViewPropertyAnimator

@see UIViewPropertyAnimator - UIKit

新的 UIViewPropertyAnimator 类可以更好地使用 UIView.animatedWithDuration API，借此可以实现很多功能，比如：

  • 中途中断动画
  • 动画开始后改变值
  • 改变 percentComplete 属性让动画更具交互性
  • 或者甚至可以把 percentComplete 和 UIGestureRecognizer 结合在一起使用

UIPasteboard

@see UIPasteboard - UIKit

UIPasteboard 现在能够 在不同的设备之间共享同一个剪贴板，例如：你在 Mac 上复制了一段话，然后粘贴到你的 iPhone/iPad 上，反过来也可以。

UITabBar

能够控制一个 tab bar item 的样式，比如背景颜色和文字属性。

ApplePay

@see WWDC 2016 - Session 703 Apple Pay on the Web

  • 苹果支付现在支持网页支付（仅限 Safari 10）
  • 支持更多新的国家

Xcode 里的新工具

@see Visual Debugging with Xcode
@see Visual Debugging With Xcode - txx 出品

  • 新的可视化调试工具（Visual Debugging Tools），特别是内存和线程工具
  • 新的代码签名机制
  • Xcode 扩展
  • 在 Xcode 8 里允许你选择使用 Swift 2.3 还是 Swift 3.0
  • 图片名字自动补全，代码中可以对颜色和图片文字化
  • 提高了 Inspector(获取iOS APP中各个控件元素的层级结构) 的可访问性，以及检查可访问性

其他

  • iPad 上会有一个 Playground App
  • 新的苹果文件系统（APFS）
  • iOS 10 开始用户能够删除苹果的内置应用（股票应用，等等）
  • 苹果开源了加密算法库和 CLI LZFSE

安全相关

@see WWDC 2016 session 706 whats new in security

  • 到 2016 年末，苹果会强制要求所有应用使用 ATS。
  • 现在你 必须 在 Info.plist 里声明 XXXUsageDescription，来获取保护的数据（例如：NSCalendarUsageDescription），如果你不这么做，你的应用在获取数据的时候会自动退出。

等等

  • Messages API
  • watchOS 3
  • tvOS

感受

  • 更加亲近Swift；
  • 更加开放；
  • 努力吸收开源社区的优点；
